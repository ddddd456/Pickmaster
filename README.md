🎮 PickMaster 7.41
PickMaster 7.41 — интерактивный гайд по героям Dota 2 для актуального патча 7.41. Помогает быстро подбирать контр-пики, изучать сборки и прокачку, а также тренировать драфт в формате «пик против случайной команды».

✨ Особенности
Каталог героев — все герои Dota 2 с фильтрацией по атрибуту и позиции.

Редактор героев — добавляй и изменяй героев (данные сохраняются в браузере). Можно загружать портреты и иконки предметов.

Тренировка пика — выбери 5 героев против случайной вражеской команды и получи оценку драфта с разбором каждого пика.

Поддержка кастомных героев — создавай своих героев с уникальными сборками, скиллбилдом, талантами и контрпиками.

Экспорт/импорт данных — выгружай свою базу героев и изображений в JSON и переноси между устройствами.

🧠 Как работает оценка драфта
При нажатии «Оценить драфт» система анализирует:

Контр-пики вражеской команды против ваших героев;

Синергию ваших героев с вражескими (если герой в списке союзников врага — это минус);

Ваши контр-пики против врагов.

Каждый пик получает оценку от 0 до 20, а общий драфт — от 0 до 100 с комментарием.

📦 Технологии
HTML, CSS, JavaScript (чистый фронтенд)

LocalStorage для хранения пользовательских данных

Drag-and-drop для загрузки изображений

Модульная структура: data.js, app.js, CSS-темы

🚀 Запуск
Клонируй репозиторий

Открой index.html в браузере (никаких сборок не требуется)

Наслаждайся!

🛠 Редактор героев
Добавление/редактирование героев с полями: ID, имя, атрибут, тип атаки, позиции, описание.

Редактирование закупа по фазам (старт, ранняя, мид, лейт, ситуативные).

Прокачка с 1 по 30 уровень (можно указать способность и заметку).

Таланты на 10/15/20/25 уровнях.

Контрпики с указанием причины.
